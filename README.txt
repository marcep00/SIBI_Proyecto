En este proyecto podemos encontrar los siguientes componentes.

- Las carpetas appbackend y appfrontend componen en gran parte la aplicación.
- El documento PDF "Base de datos" contiene el código para crear la base de datos en Neo4j.
- El documento PDF "Memoria App" corresponde, como su nombre indica, a la memoria del proyecto.
- El documento PDF "Guía de instalación" explica como instalar la aplicación para poder probarla.
- El documento PDF "Presentacion_proyecto" hace referencia a la presentación resumen del proyecto.
- Luego tenemos dos videos, uno de ellos es el de la presentación (Presentacion_Proyecto.mp4) y el otro se encuentra dividido en 3 partes en la carptea "Videos de la aplicacion", los cuales enseñan unos ejemplos de como funciona la app.
